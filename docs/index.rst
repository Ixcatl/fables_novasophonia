Landing
===================================

Welcome to the documentation page for **Us Who Remain**, a tabletop role-playing game (TTRPG) centered around small animals thriving in a post-human world.

Most core rules are derived from `the Unofficial Hollow Knight RPG <https://drive.google.com/file/d/1zWPQ9Yv7kkLrleCIlf8vSJHHxvA6-T0D/view>`__, and this project will retain the same usage rights. Personal distribution and modification is encouraged. Any commercial distribution to anyone under any circumstances is strictly prohibited.
